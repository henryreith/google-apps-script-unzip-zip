# Google Apps Script to Unzip .Zip Files in Google Drive

This repository contains a Google Apps Script that automates the process of unzipping files and creating folder structures in Google Drive. It's designed to streamline workflows for businesses and individuals who frequently work with zip files and need to maintain consistent folder structures.

## Features

- Automatically extracts contents of a .zip file into a specified Google Drive folder
- Creates a folder structure based on the contents of the zip file
- Handles nested folders and maintains the original structure
- Returns detailed information about created folders and files
- Can be integrated with external tools like Make.com for further automation

## Usage

1. Open Google Apps Script by going to script.google.com
2. Create a new project and paste the script into the editor
3. Save the project and give it a name
4. Use the `testExtractZipAndCreateFolders` function to test the script (remember to replace the file IDs with your own)

## Publishing the Script

To use this script with external tools:

1. Click on "Deploy" > "New deployment"
2. Choose "Web app" as the type
3. Set the necessary permissions
4. Deploy and save the generated URL

For detailed instructions on publishing and using the script with Make.com, refer to the blog post.
